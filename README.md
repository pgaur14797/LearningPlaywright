# 🎭 Learning Playwright with JavaScript

A hands-on learning repository for mastering **Playwright** — the modern end-to-end testing framework — using **JavaScript**.

This repo is structured chapter-by-chapter, starting from JavaScript fundamentals and progressing through core Playwright concepts.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- [Git](https://git-scm.com/)
- A code editor like [VS Code](https://code.visualstudio.com/)

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/pgaur14797/LearningPlaywright.git
   cd LearningPlaywright
   ```

2. **Verify your Node.js installation**
   ```bash
   node Chapter_01_JS_Baic/02_verify_setup.js
   ```
   This should print your platform, Node.js version, and system architecture.

3. **Run your first script**
   ```bash
   node Chapter_01_JS_Baic/01Basic.js
   ```
